# アプリの概要
- アプリの種類：ゲーム
- アプリのタイトル：記憶の迷宮

## アプリの詳細：
  - マップ全体が表示され覚える時間を設ける（簡単：15秒　普通：10秒　難しい：5秒）
  - 15x15のマップに障害物をランダムに設置
  - スタート地点に自分のキャラを設置
  - 自分の周りの1マスのみ見える（他は暗闇）
  - お助けアイテムの設置
  - 難易度別にする（覚える秒数で差別化）簡単、普通、難しい
  - 迷路はランダム生成だが正規ルートは確実に設けるようにする
  - ゲームオーバー：地雷にあたる、時間切れ

## 細かい設定

### キャラクター
- 上から見たキャラクターを表示
- サイズ：16×16px
- 男性・女性の2種類

### 障害物
- 壁
- 地雷

### お助けアイテム
- 難易度に関係なく5秒間マップ全体を表示
- 地雷によるゲームオーバーを1回だけ防ぐ

### ゲームオーバー
- 「GAME OVER」を表示
- 再スタートボタンを設置
