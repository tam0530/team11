クラス名
Map

主担当者名
油井太壱

【属性（変数）一覧】

rows : int
・マップの行数（15）

cols : int
・マップの列数（15）

cellSize : int
・1マスの大きさ（40px）

map : int[][]
・迷路全体の情報を管理する2次元配列

ROAD : final int
・通路を表す定数

WALL : final int
・壁を表す定数

MINE : final int
・地雷を表す定数

START : final int
・スタート地点を表す定数

GOAL : final int
・ゴール地点を表す定数


【メソッド一覧】

Map()
・Mapクラスのコンストラクタ。マップを初期化し、generateMap()を呼び出す。

generateMap()
・15×15の迷路を生成する。
・スタートからゴールまで必ず到達できる通路を作成する。
・通路以外に壁を配置し、さらに通路上へランダムに地雷を配置する。
・スタートとゴールには地雷を配置しない。

display()
・マップを画面に描画する。
・壁は黒色、通路は白色、スタートは「S」、ゴールは「G」、地雷は「×」として表示する。


【各属性・各メソッドに関するメモ】

rows・cols
・迷路のサイズを管理する。

cellSize
・1マスの描画サイズを管理する。

map
・各マスの状態（通路・壁・地雷・スタート・ゴール）を保存する。

ROAD
・プレイヤーが通行できるマス。

WALL
・プレイヤーが通行できない障害物。

MINE
・プレイヤーが踏むとゲームオーバーとなるマス。

START
・ゲーム開始位置。

GOAL
・ゲームクリア地点。

Map()
・ゲーム開始時にMapオブジェクトを生成する。

generateMap()
・毎回異なる迷路を生成する。
・スタートからゴールまでのルートが必ず存在するように設計する。

display()
・マップ情報をもとに画面へ描画する。
